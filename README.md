# SI Neon Nodejs API

Installation 
--
npm install

pm2 start app.js


Few important commands
--------------------------


##### - Kill instances 
* sudo killall -9 nodemon
* sudo killall -9 node

##### - Start app in background

* pm2 start app.js

##### - Restart app 

* pm2 restart app.js

##### - Delete/Stop app in background

* pm2 delete app
